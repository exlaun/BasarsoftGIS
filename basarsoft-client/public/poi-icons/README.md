# POI marker icons

These 20 white, outline SVG glyphs are the canonical POI icon assets used by the React/OpenLayers
client and packaged by the GeoServer setup script. Their filenames are the public `iconKey` values.

The glyphs are adapted from the Lucide icon family (Map Pin, Utensils, Coffee, Wheat, Heart Pulse,
Pill, Shopping Bag, Drama, Museum, Bed Double, Wrench, Circle Dollar Sign, Fuel, Bus, Plane,
Graduation Cap, Tree Pine, Trophy, Mail, and Landmark/Building). Stroke color is fixed to white so
the same file can be centered over a dynamic category-colored marker in browser and server renders.
